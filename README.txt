to get started:


1. rename the root folder (the folder containing this file) to whatever you want, ie yoursite
2. fill in your database details and the other stuff in include.php
3. load adduser.php in yoursite/admin/adduser.php in your browser and fill in your name and password
4. if you are the only user, delete adduser.php
5. load yoursite/admin in your browser and login
6. flaunt your stuff...

Documentation can be found in the docs folder in the root directory
