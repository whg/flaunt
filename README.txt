to get started:

1. fill in your database details and the other stuff in include.php
2. load adduser.php in yoursite/admin/adduser.php in your browser and fill in your name and password
3. if you are the only user, delete adduser.php
4. load yoursite/admin in your browser and login
5. flaunt your stuff...

Documentation can be found in the docs folder in the root directory
